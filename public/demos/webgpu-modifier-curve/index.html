three.js — Curve Modifier

Deform extruded text along a closed Catmull-Rom spline using GPU curve modifiers — drag control handles to reshape the path while the mesh flows along it. Based on the Three.js WebGPU curve modifier example.

<> View source code on GitHub

WebGPU required

This demo uses WebGPU curve modifiers to deform mesh geometry along a spline. Open it in a browser with WebGPU support (Chrome 113+, Edge 113+, or Firefox Nightly with WebGPU enabled).
