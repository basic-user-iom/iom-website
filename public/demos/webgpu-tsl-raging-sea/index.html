three.js — Raging Sea

Procedural ocean surface with TSL wave displacement, computed normals, and emissive crest highlights — layered large sine waves plus fractal noise. Based on the Three.js WebGPU TSL raging sea example and the Three.js Journey raging sea lesson.

<> View source code on GitHub

WebGPU required

This demo uses WebGPU node materials with TSL procedural wave displacement. Open it in a browser with WebGPU support (Chrome 113+, Edge 113+, or Firefox Nightly with WebGPU enabled).
