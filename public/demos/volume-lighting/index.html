three.js — Volumetric Lighting Rect Area

Rect area lights with volumetric fog via a post-processing pass. Import your own GLB, GLTF, or FBX from the Objects panel — that replaces the Pagani. Based on the Three.js volumetric lighting example.

<> View source code on GitHub

WebGPU required

This demo uses WebGPU volumetric lighting with rect area lights. Open it in a browser with WebGPU support (Chrome 113+, Edge 113+, or Firefox Nightly with WebGPU enabled).
