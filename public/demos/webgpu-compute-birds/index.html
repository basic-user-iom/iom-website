three.js — WebGPU compute birds

GPU flocking simulation with instanced birds — separation, alignment, and cohesion forces computed on the GPU. Move the pointer to disturb the flock. Based on the Three.js WebGPU compute birds example.

<> View source code on GitHub

WebGPU required

This demo uses WebGPU compute shaders for flocking simulation. Open it in a browser with WebGPU support (Chrome 113+, Edge 113+, or Firefox Nightly with WebGPU enabled).
