three.js — Fog Scattering

Procedural pine forest silhouettes in exponential fog with TSL density-based scattering blur — first-person walk through a cool haze. Based on the Three.js WebGPU custom fog scattering example.

<> View source code on GitHub

WebGPU required

This demo uses WebGPU render pipelines with TSL fog scattering. Open it in a browser with WebGPU support (Chrome 113+, Edge 113+, or Firefox Nightly with WebGPU enabled).
